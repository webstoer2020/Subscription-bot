# Telegram Subscription Bot

بوت تلجرام لإدارة الاشتراكات المدفوعة في القنوات.

## المميزات

- إدارة المشتركين في القنوات المدفوعة
- إشعارات تلقائية قبل انتهاء الاشتراك
- طرد تلقائي عند انتهاء الاشتراك
- واجهة إدارية سهلة
- قاعدة بيانات SQLite

## المتطلبات

- Python 3.10 أو أعلى
- Telegram Bot Token
- صلاحية إدارة في القناة (البوت يجب أن يكون مشرفاً)

## الإعداد

1. **إنشاء بوت تلجرام**:
   - افتح @BotFather في تلجرام
   - أنشئ بوت جديد واحصل على التوكن

2. **إضافة البوت للقناة**:
   - أضف البوت كمسؤول (Admin) في القناة
   - تأكد من منحه صلاحية "حذف المشاركات" و"حظر الأعضاء"

3. **تثبيت المتطلبات**:
   ```bash
   pip install -r requirements.txt
